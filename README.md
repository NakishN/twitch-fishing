# Twitch Fishing Game 🎣 (Рыбалка на Твиче)

Это локальное приложение, которое позволяет зрителям на вашем стриме ловить рыбу за баллы канала (100 баллов). Приложение автоматически создает пользовательскую награду на Twitch, слушает события выкупа в реальном времени, вычисляет улов (от обычных карасей до легендарных Кракенов!) и отображает анимированные оповещения и таблицу лидеров прямо в OBS.

## Основные возможности
* **Полная автоматизация**: Бот автоматически создает награду "Рыбалка 🎣" на вашем канале при запуске и ставит её на паузу, когда вы выключаете приложение.
* **OBS Виджет оповещений**: Анимированное всплывающее окно (поплавок раскачивается, уходит под воду с брызгами, затем показывает карту улова с описанием и цветом редкости).
* **Синтезированный звук**: Звук брызг и победные фанфары генерируются прямо в браузере с помощью Web Audio API — не нужно скачивать звуковые файлы!
* **OBS Таблица лидеров**: Прозрачный виджет для стрима, который каждые 15 секунд плавно переключается между топом по штукам, по общему весу и рекордному улову.
* **Панель управления**: Удобный веб-интерфейс для настройки, сброса статистики и симуляции тестовых уловов для проверки OBS.

## 🚀 Быстрый старт

### 1. Настройка Twitch Developer App
Поскольку приложение работает локально на вашем компьютере, вам нужно создать бесплатное приложение на портале разработчиков Twitch:

1. Перейдите на [Twitch Developer Console](https://dev.twitch.tv/console/apps) и авторизуйтесь под своим стримерским аккаунтом.
2. Нажмите **Register Your Application** (Зарегистрировать приложение).
3. Укажите следующие данные:
   * **Name**: `Twitch Fishing <ваш_ник>` (название должно быть уникальным на Twitch).
   * **OAuth Redirect URLs**: `http://localhost:3000/auth/callback` (обязательно нажмите **Add**).
   * **Category**: Выберите `Application Integration` или `Other`.
   * **Client Type**: `Confidential`.
4. Нажмите **Create** (Создать).
5. В списке приложений найдите созданное и нажмите **Manage** (Управление).
6. Скопируйте **Client ID**.
7. Нажмите **New Secret** (Новый секрет) и скопируйте сгенерированный **Client Secret** (сохраните его, больше его не покажут).

### 2. Запуск сервера рыбалки

Откройте терминал и перейдите в папку проекта:
```bash
cd /home/nwk/Рабочий стол/twitch-fishing
./run.sh
```

Сервер запустится на порту `3000`. 
Откройте в браузере панель управления: **[http://localhost:3000](http://localhost:3000)**.

### 3. Авторизация канала

1. В панели управления вставьте ваши **Client ID** и **Client Secret**.
2. Нажмите кнопку **💾 Сохранить настройки**.
3. Нажмите появившуюся фиолетовую кнопку **🔌 Авторизоваться через Twitch** и подтвердите доступ для вашего канала.
4. После перенаправления обратно включите переключатель **Активность награды**.

Награда автоматически появится в вашем списке наград на Twitch!

## 📺 Настройка в OBS Studio

Добавьте оверлеи в OBS в качестве источников типа **Браузер** (Browser Source):

### 1. Виджет оповещений (Анимация улова)
* **Ссылка**: `http://localhost:3000/widget.html`
* **Ширина**: `1920`
* **Высота**: `1080`
* **Управление звуком через OBS**: Рекомендуется поставить галочку, чтобы регулировать громкость виджета в микшере OBS.

### 2. Виджет таблицы лидеров (Leaderboard)
* **Ссылка**: `http://localhost:3000/leaderboard.html`
* **Ширина**: `380`
* **Высота**: `360`
* Разместите виджет в удобном углу экрана. Он прозрачен и плавно обновляется при каждом новом улове.

*Используйте форму **Тестирование и отладка** в панели управления, чтобы отправить тестовый улов и настроить расположение виджетов.*
