⚙️ Настройки API

Инструкция:

  1. Зарегистрируйте приложение на Twitch Developer Console.
  2. В поле OAuth Redirect URLs впишите: http://localhost:3000/auth/callback
  3. Скопируйте Client ID и Client Secret в поля выше, сохраните настройки и авторизуйтесь.

📺 Ссылка для OBS (Браузер)

http://localhost:3000/widget.html
http://localhost:3000/leaderboard.html

Добавьте эти ссылки как «Браузер» (Browser Source) в OBS Studio. Размер оверлея оповещений рекомендуется: 1920x1080 (прозрачный).
